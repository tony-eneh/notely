# PWA Icons Setup

The PWA is configured and requires the following icon files in this directory:

## Required Icons:
- `icon-192x192.png` - Standard icon (192x192px)
- `icon-512x512.png` - High-res icon (512x512px)
- `icon-192x192-maskable.png` - Maskable icon (192x192px with safe zone)
- `icon-512x512-maskable.png` - Maskable icon (512x512px with safe zone)

## Optional but Recommended:
- `screenshot-wide.png` - Desktop screenshot (1280x720px)
- `screenshot-narrow.png` - Mobile screenshot (750x1334px)

## Tools for Creating Icons:
1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
2. **Favicon.io**: https://favicon.io/
3. **Maskable.app**: https://maskable.app/editor (for maskable icons)

## Maskable Icon Guidelines:
- Keep important content within the central 80% "safe zone"
- Use solid background color (not transparent)
- Test with Maskable.app editor

## Quick Setup:
1. Create a 512x512px logo
2. Upload to PWA Asset Generator
3. Download all generated icons
4. Place in this directory

Until icons are added, the PWA will use default browser icons.
